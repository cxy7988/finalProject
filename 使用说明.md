# 课程与教授评分系统 - 使用说明

## 项目结构

```
finalProject/
├── Rating.java              # 评分类
├── Professor.java           # 教授类
├── CourseProfessor.java     # 课程-教授关系类
├── Course.java              # 课程类
├── RatingSystem.java        # 系统核心类
├── Main.java                # 主程序入口
├── data.csv                 # 示例数据文件
└── README.md                # 项目文档
```

## 编译与运行

### 方法一：使用命令行

1. 进入src目录的父目录：
```bash
cd /Users/cxy/javaProjects/myJavaCourse/src
```

2. 编译所有Java文件：
```bash
javac CPS2232/finalProject/*.java
```

3. 运行程序：
```bash
java CPS2232.finalProject.Main
```

### 方法二：使用IDE（推荐）

1. 在IntelliJ IDEA或Eclipse中打开项目
2. 找到Main.java文件
3. 右键点击，选择"Run 'Main.main()'"

## 功能说明

### 1. 从文件加载数据
- 选择菜单选项 1
- 输入CSV文件名（例如：data.csv）
- 系统会自动解析并加载所有数据

### 2. 添加新课程
- 选择菜单选项 2
- 依次输入课程编号和课程名称
- 课程会被添加到系统中

### 3. 添加新评价
- 选择菜单选项 3
- 依次输入：
  - 课程编号
  - 课程名称
  - 教授姓名
  - 评分（0-5分）
  - 评论内容

### 4. 按课程ID查询（二分查找）
- 选择菜单选项 4
- 输入完整的课程编号（例如：CPS1231）
- 显示课程详细信息和所有教授评价

### 5. 按课程名查询（线性查找）
- 选择菜单选项 5
- 输入课程名关键字（例如：Java）
- 显示所有匹配的课程列表
- 可选择查看某门课程的详细信息

### 6. 按教授姓名查询
- 选择菜单选项 6
- 输入教授完整姓名（例如：Dr. Smith）
- 显示该教授的所有授课信息和评价

### 7. 显示教授排名（选择排序）
- 选择菜单选项 7
- 选择排名方式：
  - 全局教授排名：所有教授按总体平均分排序
  - 某门课程内教授排名：特定课程的教授评分排名

### 8. 保存数据到文件
- 选择菜单选项 8
- 输入保存的文件名（例如：output.csv）
- 所有数据会以CSV格式保存

### 0. 退出程序
- 选择菜单选项 0
- 程序退出

## CSV文件格式

```csv
courseId,courseName,professorName,rating,comment
CPS1231,Java Programming,Dr. Smith,4.5,Very clear explanation
MATH2413,Calculus I,Dr. Lee,3.8,Homework is heavy
```

每行包含：
- courseId: 课程编号
- courseName: 课程名称
- professorName: 教授姓名
- rating: 评分（浮点数，建议0-5）
- comment: 评论内容

## 核心算法实现

### 1. 线性查找（Linear Search）
- 位置：`RatingSystem.searchCoursesByName()`
- 用途：按课程名关键字模糊搜索
- 时间复杂度：O(n)

### 2. 二分查找（Binary Search）
- 位置：`RatingSystem.searchCourseById()`
- 用途：按课程ID精确查找
- 前提：课程列表按ID排序
- 时间复杂度：O(log n)

### 3. 选择排序（Selection Sort）
- 位置：`RatingSystem.selectionSortByRating()` 和 `selectionSortProfessors()`
- 用途：
  - 对课程内教授按评分排序
  - 对全局教授按评分排序
- 时间复杂度：O(n²)

## 数据结构使用

### HashMap<String, Course> courseMap
- 用途：按课程ID快速查找课程
- 查找复杂度：O(1)

### ArrayList<Course> courses
- 用途：遍历、排序、二分查找
- 支持全局操作

### List<CourseProfessor>
- 用途：存储每门课程的教授列表
- 支持选择排序

### List<Rating>
- 用途：存储每个课程-教授组合的所有评分
- 支持平均分计算

## 示例操作流程

1. 启动程序
2. 选择"1"加载data.csv文件
3. 选择"4"按课程ID查询"CPS1231"
4. 查看该课程的教授排名和评价
5. 选择"7"查看全局教授排名
6. 选择"3"添加新评价
7. 选择"8"保存数据到output.csv
8. 选择"0"退出

## 注意事项

1. 确保CSV文件格式正确，使用逗号分隔
2. 评分建议使用0-5的范围
3. 教授姓名和课程ID需要精确匹配（区分大小写）
4. 课程名查询支持模糊匹配（不区分大小写）
5. 保存文件前请确认文件名，避免覆盖重要数据
